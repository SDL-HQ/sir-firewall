# Proof artifact roles (D2 hygiene)

This directory intentionally contains multiple artifact roles:

1. **Current working pointers (mutable)**
   - `latest-audit.json`, `latest-audit.html`
   - `run_summary.json`, `itgl_ledger.jsonl`, `itgl_final_hash.txt`, `latest-attempts.log`
   - `run_id.txt`

2. **Archived per-run truth (immutable per run)**
   - `runs/<run_id>/` (`audit.json`, `manifest.json`, `archive_receipt.json`, copied run proofs)

3. **Legacy retained material**
   - historical pre-2.0 / transitional files kept for traceability
   - includes historical certificate files that remain in their original locations

4. **Static/template/helper assets**
   - `template.html`
   - `archive/` for archival certificate snapshots generated by current tooling

Notes:
- Default current review should use current pointers and `runs/` indexes, not legacy leftovers.
- Historical files are retained; this document does not reinterpret their semantics.
